# qa-copilot
AI-powered QA assistant that generates structured test cases from software requirements using Gemini AI and Streamlit.

## Offline / Demo mode
If your Gemini API key is limited or unavailable, the app can run in demo mode instead.

- Set `QA_COPILOT_DEMO=1` before starting the app to force demo mode.
- If `GOOGLE_API_KEY` is not set, the app will also fall back to demo mode automatically.
